#  About FinBank:
# FinBank<br>Banking Transaction Management System<br><br>FinBank is a full-stack banking application that allows users to register, log in, transfer money between accounts, view transaction history, and check account balance.<br><br>The system is built using:<br><br>Spring Boot for the backend REST API<br><br>React for the frontend interface<br><br>MySQL as the database<br><br>Bootstrap for UI styling<br><br>Features<br>User Authentication<br><br>Register new users<br><br>Login functionality<br><br>Navbar dynamically updates after login<br><br>Protected routes using private route logic<br><br>Account Management<br><br>Every new account starts with ₹1000 initial balance<br><br>Users can check their account balance<br><br>Money Transfer<br><br>Transfer money between accounts<br><br>Sender and receiver must be different<br><br>Sender balance decreases<br><br>Receiver balance increases<br><br>Validation for insufficient balance<br><br>Transaction History<br><br>Users can view transaction records where they are:<br><br>Sender (fromAccount)<br><br>Receiver (toAccount)<br><br>Dashboard<br><br>Displays user profile information:<br><br>User ID<br><br>Name<br><br>Email
